조동사 시제별/수동태 완전 정리

📌 사용 설명: 이 문서는 주요 조동사의 모든 시제 형태와 수동태 용법을 체계적으로 정리하였습니다. 각 조동사별로 기본 용법부터 다양한 시제와 수동태 형태까지 확인할 수 있습니다.

1. Will

시제별/수동태 용법

형태예문해석
Will + 동사원형I will call you later.나중에 전화할게요.
Will + be + -ingI will be working at 3 PM.나는 오후 3시에 일하고 있을 거예요.
Will + be + p.p. (수동태)The room will be cleaned soon.방은 곧 청소될 거예요.
Will + have + p.p.She will have finished it by then.그녀는 그때까지 끝냈을 거예요.
Will + have been + p.p. (수동태)The report will have been submitted by Friday.보고서는 금요일까지 제출됐을 거예요.

2. Can

시제별/수동태 용법

형태예문해석
Can + 동사원형She can speak French.그녀는 프랑스어를 할 수 있어요.
Can + be + -ingHe can be working now.그는 지금 일하고 있을 수 있어요.
Can + be + p.p. (수동태)This rule can be changed.이 규칙은 변경될 수 있어요.
Could + 동사원형 (과거)I could swim when I was young.나는 어렸을 때 수영할 수 있었어요.
Could + have + p.p.You could have told me earlier.너는 나에게 더 일찍 말할 수 있었어.

3. Must

시제별/수동태 용법

형태예문해석
Must + 동사원형You must finish your homework.너는 숙제를 끝내야 해.
Must + be + -ingHe must be sleeping now.그는 지금 자고 있을 거야.
Must + be + p.p. (수동태)The documents must be signed.서류들은 서명되어야 해.
Must + have + p.p.She must have left already.그녀는 이미 떠났을 거야.
Must + have been + p.p. (수동태)The door must have been locked.문은 잠겼었을 거야.

4. Should

시제별/수동태 용법

형태예문해석
Should + 동사원형You should see a doctor.너는 의사를 봐야 해.
Should + be + -ingYou should be studying now.너는 지금 공부하고 있어야 해.
Should + be + p.p. (수동태)This matter should be discussed.이 문제는 논의되어야 해.
Should + have + p.p.You should have apologized.너는 사과했어야 했어.
Should + have been + p.p. (수동태)The letter should have been sent.편지는 보내졌어야 했어.

5. May/Might

시제별/수동태 용법

형태예문해석
May + 동사원형It may rain tomorrow.내일 비가 올지도 몰라요.
May + be + -ingShe may be waiting for us.그녀는 우리를 기다리고 있을지도 몰라요.
May + be + p.p. (수동태)The keys may be found soon.열쇠는 곧 발견될지도 몰라요.
May + have + p.p.He may have forgotten.그는 잊었을지도 몰라요.
May + have been + p.p. (수동태)The package may have been delivered.소포는 배달되었을지도 몰라요.
Might + 동사원형It might snow tonight.오늘 밤 눈이 올지도 몰라요.
Might + be + -ingShe might be sleeping now.그녀는 지금 자고 있을지도 몰라요.
Might + be + p.p. (수동태)The decision might be changed.결정이 바뀔지도 몰라요.
Might + have + p.p.She might have missed the train.그녀는 기차를 놓쳤을지도 몰라요.
Might + have been + -ingHe might have been working then.그는 그때 일하고 있었을지도 몰라요.
Might + have been + p.p. (수동태)The letter might have been sent already.편지는 이미 보내졌을지도 몰라요.

6. Would

시제별/수동태 용법

형태예문해석
Would + 동사원형I would travel if I had money.내가 돈이 있다면 여행할 텐데.
Would + be + -ingShe would be working now.그녀는 지금 일하고 있을 텐데.
Would + be + p.p. (수동태)The problem would be solved.문제는 해결될 텐데.
Would + have + p.p.I would have helped you.내가 너를 도왔을 텐데.
Would + have been + p.p. (수동태)The mistake would have been noticed.실수는 눈치채졌을 텐데.

7. Could

시제별/수동태 용법

형태예문해석
Could + 동사원형I could swim when I was young.나는 어렸을 때 수영할 수 있었어요.
Could + be + -ingHe could be joking.그는 농담하고 있을 수도 있어요.
Could + be + p.p. (수동태)This could be done differently.이것은 다르게 처리될 수 있어요.
Could + have + p.p.You could have told me.너는 나에게 말할 수 있었어.
Could + have been + -ingShe could have been waiting.그녀는 기다리고 있었을 수도 있어.
Could + have been + p.p. (수동태)It could have been prevented.그것은 예방될 수 있었어.

8. 조동사 + have + 과거분사 완전 정리

주요 용법과 예시

형태의미예문해석
Must have p.p.과거에 틀림없이 ~했을 것이다She must have known the truth.그녀는 진실을 알았을 거야.
Should have p.p.~했어야 했는데 (후회/비판)You should have studied harder.너는 더 열심히 공부했어야 했어.
Could have p.p.~할 수 있었는데 (안 함)I could have won the game.나는 게임에서 이길 수 있었어.
May have p.p.~했을지도 모른다He may have left already.그는 이미 떠났을지도 몰라.
Might have p.p.~했을지도 모른다 (가능성 더 낮음)She might have forgotten.그녀는 잊었을지도 몰라.
Would have p.p.~했을 텐데 (가정)I would have called you.내가 너에게 전화했을 텐데.
Can't have p.p.~했을 리가 없다He can't have stolen it.그가 그것을 훔쳤을 리가 없어.
💡 중요: '조동사 + have + 과거분사' 구조는 모두 과거에 대한 추측, 후회, 가능성 등을 나타냅니다. 각 조동사마다 미묘한 의미 차이가 있으니 주의해서 구분해야 합니다.

9. Used to

시제별 용법

형태예문해석
Used to + 동사원형I used to play soccer every weekend.나는 매주 주말에 축구를 하곤 했어요.
Didn't use to + 동사원형I didn't use to like coffee.나는 원래 커피를 좋아하지 않았어요.
Did...use to + 동사원형?Did you use to live here?너 원래 여기 살았어?
Be used to + 명사/-ingI'm used to waking up early.나는 일찍 일어나는 것에 익숙해요.
Get used to + 명사/-ingYou'll get used to the noise.너는 그 소음에 익숙해질 거야.
✅ 최종 확인 사항: 이 문서는 Will, Can, Must, Should, May/Might, Would, Could, Used to 등 주요 조동사의 모든 시제와 수동태 형태를 포함하고 있습니다. Might 부분에 누락되었던 형태들도 모두 추가되었으며, 가독성을 높이기 위한 디자인 개선도 적용되었습니다.
